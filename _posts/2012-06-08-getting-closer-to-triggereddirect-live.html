We've been working on a live set up where we could use the sounds and effects that are more like what is actually on our albums. So here is a test run of our song Voomp Boomp.

Rob created a triggered drum set mounting them inside and replacing the heads with mesh. We run these through a midi brain that plays sound samples in Renoise on an ASUS netbook running AVLinux. And all the other instruments and vocals run direct through a mixing board into ProTools, with the effects we used from the original recording. We were pretty happy with the outcome.

Watch this triggered/direct live sound set up video on YouTube, and while you're there, subscribe to our channel.
